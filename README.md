# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'Carbon.SmbShare'
Import-Module -Name 'Carbon.SmbShare'
```

To install privately:

```powershell
Save-Module -Name 'Carbon.SmbShare' -Path '.'
Import-Module -Name '.\Carbon.SmbShare'
```
